## Options

The fastest way to get started with **nuxt-module-template** is to define the options:

```js {}[nuxt.config.js]
{
  modules: [
    ['nuxt-module-template', {
      ...
    }]
  ],
}
```

*We strongly recommend to use environment variables for all these values!*
